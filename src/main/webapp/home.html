Leave Approval System
This is our solution to the quintessential leave management problem. This system offers a simple yet elegant UI that promotes good workplace practices. Through this system, we aim to reduce the clutter created by piles of paperwork. Privacy features implemented within the core functionalities of the system keep the users' data secure. There are two kinds of users in this system. The first one is a "manager". A manager has responsibilities that are analogous to that of an administrator. The second one is an "employee". An employee actively takes part by making leave or account creation requests which will then be forwarded to and handled by the managers.
